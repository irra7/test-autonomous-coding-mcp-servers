# test-autonomous-coding MCP Servers

Custom MCP servers for the agentic coding framework.

## Overview

Este repositorio contiene MCP servers custom para el proyecto test-autonomous-coding.

**Nota**: Por ahora, usamos los MCP servers públicos:
- `@modelcontextprotocol/server-github` - GitHub integration

## Servers Disponibles

### GitHub MCP (público)
Configurado en Claude Desktop

### Railway MCP (futuro)
Para auto-deploy y gestión de Railway

### AWS MCP (futuro)
Para integración con servicios AWS del proyecto BI
